Detailed Description

template<typename T>
class CarWashQueue< T >

Шаблонный класс для управления очередью машин.

Класс предоставляет методы для добавления машин, удаления машины из очереди (обслуживания) и проверки размера очереди.

Template Parameters
TТип данных, хранимых в очереди. Например, std::string для хранения названий машин.
